# Security Policy

## Protecting credentials

- Do **not** commit API keys, tokens, or secrets.
- Store credentials in environment variables (for example: `ANTHROPIC_API_KEY`, `OPENAI_API_KEY`).
- Scrub example configs and logs before sharing.

## Reporting vulnerabilities

Please report security vulnerabilities privately via GitHub Security Advisories for this repository. If that is not available, open an issue with minimal details and we will follow up to establish a private channel.
